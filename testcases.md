# [Webull](https://www.webull.com/) test cases
## 1. Размещение заказа на акции (достаточное число денег)
### Предусловия:
- создан аккаунт
- открыта вкладка trade на сайте
### Выполнение
| Шаг | Ожидаемый результат |
| --- | ------------------- |
| Перейти к панели купли-продажи | Панель становится активной |
| Выбрать опцию buy | Опция buy ставновится активной |
| Для поля Quantity в выпадающем списки выбрать денежное выражение приобретаемых акций | Перед значением в поле Quantity появился значок "$" |
| Ввести сумму меньшую, чем имеется на счету, нажать trade | На панели торгов, вкладка working появляется новый заказ на заданную суммую |

## 5. Создание уведомлений о достижении заданной цены

- создан аккаунт
- открыта вкладка trade на сайте
### Выполнение
| Шаг | Ожидаемый результат |
| --- | ------------------- |
| Перейти к панели с графиком цены | Панель становится активной |
| Нажать ПКМ по графику | Открывается контекстное меню |
| Выбрать опцию create alert | Появляется всплывающее окно |
| Ввести нужную сумму, нажать save | При достижении заданной цены появляется окошко с уведомлением |

## 6. Отмена активного торга
### Предусловия:
- создан аккаунт
- открыта вкладка trade на сайте
- размещен активный торг на акции
### Выполнение
| Шаг | Ожидаемый результат |
| --- | ------------------- |
| Перейти к панели с торгами | Панель становится активной |
| Перейти на вкладку working | Открывается список активных торгов |
| ПКМ по одному из активных торгов | Появится контекстное меню |
| Выбрать опцию cancel trade | Торг перемещается из вкладки working во вкладку canceled |

## 10. Добавление акций компании в список отслеживаемых
- создан аккаунт
- открыта вкладка trade на сайте
- активна панель с графиком цен на акции компании, которая не находится в списке отслеживаемых
### Выполнение
| Шаг | Ожидаемый результат |
| --- | ------------------- |
| Нажать ПКМ по графику | Открывается контекстное меню |
| Выбрать add to watchlist | Открывается набор имеющихся списков отслеживания |
| Выбрать один из списков | На панели списков отслеживания в выбранном списке акции нужной компании |
